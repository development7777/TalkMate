## TalkMateとは

TalkMateは、人と会話をし、共感するスマートロボットです。 自然な会話を通じて、ユーザーの日常生活をサポートし、伴走してくれます。そして、一人ひとりのニーズに合わせた柔軟なコミュニケーションを実現します。

## 機能

TalkMateは、次の機能はあります。

- [ ] Arduinoで駆動
- [ ] 人と会話する
- [ ] HTTPサーバーとして機能し、Webブラウザで操作できる
- [ ] ホームベースで自動充電・アップデートする
