## What is TalkMate?

TalkMate is a smart robot that converses with and empathizes with people. Through natural conversation, it supports users' daily lives and accompanies them. It achieves flexible communication tailored to each individual's needs.

## Features

TalkMate has the following features:

- [ ] Arduino-powered
- [ ] Converses with people
- [ ] Functions as an HTTP server, operable via web browser
- [ ] Automatically charges and updates at home base
